jQuery.cgClassPopup (options)

描述：弹出框 demo

继承自： Modernizr, jQuery

源码： jquery.plugin.popup.js

作者：徐晨

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • addClass {css class} 附加弹出框样式
  • append {Object} 以添加的方式初始化弹出框 isAppend: 是否以添加的方式初始化弹出框; dom: 包裹对象
  • autoShow {Boolean} 是否在初始化时自动显示
  • cancel {Function} 取消按钮附加函数
  • cancelText {String} 取消按钮的文字
  • content {Function} 内容区的附加函数
  • done {Function} 确定按钮附加函数
  • doneText {String} 确定按钮的文字
  • hasBtn {Boolean} 是否需要按钮
  • hasCancel {Boolean} 是否需要取消按钮
  • isCenter {Boolean} 是否居中
  • isDrag {Boolean} 是否唯一
  • isLayer {Boolean} 是否需要遮罩层
  • isOnly {Boolean} 是否唯一
  • message {String} 弹出框为展示文本类型时显示的消息提示文本
  • param {Object} 附加参数
  • popupTemp {jQuery Dom} 弹出框html的jQuery对象
  • template {html String} 内容区的html代码片段
  • time {Number} 如果type = result 自动消失时间
  • title {String} 弹出框的标题
  • type {String} 弹出框的种类 [ "popup" | "confirm" | "result" ]

Static Properties

Events

  • bindDrag () 绑定拖拽事件
  • close (callback) 关闭弹出框
  • disableBtn (isReset) 改变确定按钮状态
  • hide (callback) 隐藏弹出框
  • positionCenter () 使弹出框居中
  • removeTip () 隐藏错误信息
  • reset (newHtml) 重置内容区内容
  • show (callback) 显示弹出框
  • showTip (str) 显示错误信息

  • jQuery.cgClassPopup (options)
    弹出框 demo
    参数：
    •   options {Object} 参数对象数据
    •   options.type {String} 弹出框的种类 [ "popup" | "confirm" | "result" ]
    •   options.param {Object} 附加参数
    •   options.title {String} 弹出框的标题
    •   options.popupTemp {jQuery Object} 弹出框html的jQuery对象
    •   options.template {html} 内容区的html代码片段
    •   options.message {String} type为confirm时显示的消息提示文本
    •   options.time {Number} type为result时自动关闭窗口时间
    •   options.addClass {css class} 附加弹出框样式
    •   options.isLayer {Boolean} 是否需要遮罩层
    •   options.isCenter {Boolean} 是否居中
    •   options.isDrag {Boolean} 是否可拖拽
    •   options.isOnly {Boolean} 是否为唯一
    •   options.autoShow {Boolean} 是否在初始化时自动显示弹出框
    •   options.append {Object} 是否在目标对象中加载
    •   options.append.isAppend {Boolean} 是否在目标对象中加载
    •   options.append.dom {jQuery Object} 目标对象的jquery dom
    •   options.hasBtn {Boolean} 是否需要按钮
    •   options.hasCancel {Boolean} 是否需要取消按钮
    •   options.content {Function} 内容区的附加函数
    •   options.done {Function} 确定按钮的附加函数
    •   options.cancel {Function} 取消按钮的附加函数

Properties Detail

  • addClass {css class} — 附加弹出框样式
    默认值：""
  • append {Object} — 以添加的方式初始化弹出框 isAppend: 是否以添加的方式初始化弹出框; dom: 包裹对象
    默认值：isAppend : false, dom : $(".dom")
  • autoShow {Boolean} — 是否在初始化时自动显示
    默认值：true
  • cancel {Function} — 取消按钮附加函数
    默认值：null
  • cancelText {String} — 取消按钮的文字
    默认值："取消"
  • content {Function} — 内容区的附加函数
    默认值：null
  • done {Function} — 确定按钮附加函数
    默认值：null
  • doneText {String} — 确定按钮的文字
    默认值："确定"
  • hasBtn {Boolean} — 是否需要按钮
    默认值：true
  • hasCancel {Boolean} — 是否需要取消按钮
    默认值：true
  • isCenter {Boolean} — 是否居中
    默认值：true
  • isDrag {Boolean} — 是否唯一
    默认值：true
  • isLayer {Boolean} — 是否需要遮罩层
    默认值：true
  • isOnly {Boolean} — 是否唯一
    默认值：false
  • message {String} — 弹出框为展示文本类型时显示的消息提示文本
    默认值：""
  • param {Object} — 附加参数
  • popupTemp {jQuery Dom} — 弹出框html的jQuery对象
    默认值：null
  • template {html String} — 内容区的html代码片段
    默认值：""
  • time {Number} — 如果type = result 自动消失时间
    默认值：null
  • title {String} — 弹出框的标题
    默认值：""
  • type {String} — 弹出框的种类 [ "popup" | "confirm" | "result" ]
    默认值："popup"

    •   callback {Function} 回调函数

    显示错误信息
