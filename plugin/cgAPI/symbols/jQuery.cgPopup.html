jQuery.cgPopup (options)

描述：弹出框

继承自： jQuery

源码： jquery.plugin.popup.js

作者：徐晨

Events

  • close (callback) 关闭弹出框
  • disableBtn (disable) 改变确定按钮状态
  • removeTip () 隐藏错误信息
  • showTip (str) 显示错误信息

Class Detail

  • jQuery.cgPopup (options)
    弹出框
    参数：
    •   options {Object} 参数对象数据
    •   options.title {String} 弹出框的标题
    •   options.popupTemp {jQuery Object} 弹出框html的jQuery对象
    •   options.template {html} 内容区的html代码片段
    •   options.addClass {css class} 附加弹出框样式
    •   options.isLayer {Boolean} 是否需要遮罩层
    •   options.isCenter {Boolean} 是否居中
    •   options.isDrag {Boolean} 是否可拖拽
    •   options.isOnly {Boolean} 是否为唯一
    •   options.append {Object} 是否在目标对象中加载
    •   options.append.isAppend {Boolean} 是否在目标对象中加载
    •   options.append.dom {jQuery Object} 目标对象的jquery dom
    •   options.hasBtn {Boolean} 是否需要按钮
    •   options.hasCancel {Boolean} 是否需要取消按钮
    •   options.content {Function} 内容区的附加函数
    •   options.done {Function} 确定按钮的附加函数
    •   options.cancel {Function} 取消按钮的附加函数
    Since
    version 1.0
    示例：
    $.cgPopup(
    {
  [可选]title : "弹出框",
  [可选]popupTemp : null || $('
    '+
                    ' '+
                    '
    ' + option.title + '
    '+
                    '
    '+
                    '
    '+
                        '确认'+
                        '取消'+
                    '
    '+
                '
    '),
        template : "
    ",
  [可选]addClass : "popupClass",
  [可选]isLayer : true,
  [可选]isCenter : true,
  [可选]isDrag : true,
  [可选]isOnly : false,
  [可选]append : {
            isAppend : false,
            dom : $(".dom")
        },
  [可选]hasBtn : true,
  [可选]hasCancel : true,
  [可选]content : null || function(opt){},
  [可选]done : null || function(opt){},
  [可选]cancel : null || function(opt){}
    }
);
