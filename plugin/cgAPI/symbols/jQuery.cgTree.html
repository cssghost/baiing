jQuery.cgTree (options)

描述：自动完成插件 demo

继承自： jQuery, jQuery.getFromTemplate

源码： jquery.plugin.tree.js

作者：徐晨

Events

  • addItem (item) 添加一个节点
  • closeSelected (id, prefix) 关闭当前节点
  • openRoot () 打开所有顶级节点，并关闭其他节点
  • openSelected (id, prefix) 打开当前节点

Class Detail

  • jQuery.cgTree (options)
    自动完成插件 demo
    参数：
    •   options {Object} 参数对象数据
    •   options.wrap {jQuery Object} 包裹的jquery dom
    •   options.treeData {json} 树形数据 {"id":"id","pId":"pId|null","name":"描述型知识"(,"prefix":"c")}
    •   options.addClass {css class} 点击节点时需要添加的样式
    •   options.isPreFix {Boolean} 是否存在特殊识别符
    •   options.template {jQuery Object} 节点模板对象（按照jQuery.getFromTemplate方法）
    •   options.nodeFun {Function} 点击节点时的附加事件
    •   options.addOtherFun {Function} 添加非末级节点时的附加事件
    •   options.addLastFun {Function} 添加末级节点时的附加事件
    •   options.callback {Function} 模块初始化时的附加事件
    Since
    version 1.0
    示例：
    $.cgTree({
        wrap : $(".Js-has-prefix"),
        treeData : [{"id":"c102","pId":"c97","prefix":"c","name":"描述型知识","type":"catalog"}],
        template : $("#Js-tree-item-template"),
        isPreFix : true,
        callback : function (opt) {
        }
    });
